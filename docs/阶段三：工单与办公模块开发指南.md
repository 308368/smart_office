### 3.1.4 工单管理接口

| 接口 | 方法 | 描述 | 权限 |
|------|------|------|------|
| /office/ticket/create | POST | 创建工单 | - |
| /office/ticket/list | GET | 工单列表 | - |
| /office/ticket/{id} | GET | 工单详情 | - |
| /office/ticket/{id}/handle | PUT | 处理工单 | ticket:handle |
| /office/ticket/{id}/transfer | PUT | 转派工单 | ticket:transfer |
| /office/ticket/{id}/close | PUT | 关闭工单 | - |
| /office/ticket/{id}/reply | POST | 回复工单 | - |
| /office/ticket/type/list | GET | 工单类型列表 | - |

```java
@RestController
@RequestMapping("/office/ticket")
@RequiredArgsConstructor
public class TicketController {

    private final TicketService ticketService;

    @PostMapping("/create")
    public Result<TicketVO> create(@RequestBody TicketDTO dto) {
        return Result.success(ticketService.create(dto));
    }

    @GetMapping("/list")
    public Result<PageResult<TicketVO>> list(
            @RequestParam(defaultValue = "1") Long current,
            @RequestParam(defaultValue = "10") Long size,
            @RequestParam(required = false) Integer status,
            @RequestParam(required = false) Long typeId,
            @RequestParam(required = false) String title
    ) {
        return Result.success(ticketService.listTicket(current, size, status, typeId, title));
    }

    @GetMapping("/{id}")
    public Result<TicketVO> getById(@PathVariable Long id) {
        return Result.success(ticketService.getById(id));
    }

    @PutMapping("/{id}/handle")
    @RequiresPermissions("ticket:handle")
    public Result<Void> handle(@PathVariable Long id, @RequestBody TicketDTO dto) {
        ticketService.handle(id, dto);
        return Result.success();
    }

    @PutMapping("/{id}/close")
    public Result<Void> close(@PathVariable Long id) {
        ticketService.close(id);
        return Result.success();
    }

    @PostMapping("/{id}/reply")
    public Result<Void> reply(@PathVariable Long id, @RequestBody TicketDTO dto) {
        ticketService.reply(id, dto);
        return Result.success();
    }

    @GetMapping("/type/list")
    public Result<List<TicketTypeVO>> typeList() {
        return Result.success(ticketService.getTypeList());
    }
}
```

### 3.2.3 后端Controller实现

```java
@RestController
@RequestMapping("/office/leave")
@RequiredArgsConstructor
public class LeaveController {

    private final LeaveService leaveService;

    @PostMapping("/create")
    @RequiresPermissions("leave:add")
    public Result<LeaveVO> create(@RequestBody @Valid LeaveDTO dto) {
        return Result.success(leaveService.create(dto));
    }

    @GetMapping("/list")
    @RequiresPermissions("leave:list")
    public Result<PageResult<LeaveVO>> list(
            @RequestParam(defaultValue = "1") Long current,
            @RequestParam(defaultValue = "10") Long size,
            @RequestParam(required = false) Integer status,
            @RequestParam(required = false) String leaveType
    ) {
        return Result.success(leaveService.listByUser(current, size, status, leaveType));
    }

    @GetMapping("/pending")
    @RequiresPermissions("leave:approve")
    public Result<PageResult<LeaveVO>> pending(
            @RequestParam(defaultValue = "1") Long current,
            @RequestParam(defaultValue = "10") Long size
    ) {
        return Result.success(leaveService.listPending(current, size));
    }

    @GetMapping("/balance")
    public Result<Map<String, Object>> balance() {
        return Result.success(leaveService.getBalance());
    }

    @PutMapping("/approve/{id}")
    @RequiresPermissions("leave:approve")
    public Result<Void> approve(@PathVariable Long id, @RequestBody @Valid ApproveDTO dto) {
        leaveService.approve(id, dto);
        return Result.success();
    }

    @PutMapping("/cancel/{id}")
    @RequiresPermissions("leave:cancel")
    public Result<Void> cancel(@PathVariable Long id) {
        leaveService.cancel(id);
        return Result.success();
    }
}
```


#### 审批页面功能

1. **待审批列表**
   - 显示所有待审批的请假申请
   - 筛选条件：请假类型、申请人、部门
   - 显示内容：申请人、请假类型、日期范围、天数、状态、申请时间

2. **审批操作**
   - 同意：更新状态为"已通过"，记录审批时间和审批人
   - 拒绝：更新状态为"已拒绝"，记录审批意见
   - 弹窗填写审批意见

3. **权限控制**
   - 仅具有 `leave:approve` 权限的用户可见"请假审批"菜单
   - 仅 `status=1待审批` 的申请可审批
